## Project Overview

Libellule Senteurs — luxury home fragrance brand showcase website. This is a **presentation/vitrine site**, NOT e-commerce. The goal is to convey premium luxury (think Diptyque, Byredo, Le Labo level). Primary language: French.

## Tech Stack

- **Next.js 14+** (App Router) with TypeScript
- **Tailwind CSS** for base styling
- **GSAP** (ScrollTrigger, SplitText, Draggable, InertiaPlugin) — primary animation engine
- **Lenis** — smooth scroll
- **Framer Motion** — React component animations (mount/unmount, layout, hover)
- **Remotion** — pre-rendered animation sequences (loading screen, hero mist, product morph image sequence)
- **Three.js / React Three Fiber** — optional for golden particles and 3D effects

## Design System

### Colors (use sparingly — gold accents, black/white dominant)
| Token | HEX | Usage |
|-------|-----|-------|
| Or Luxe | `#C99700` | Signature accent, logo, premium elements — use with restraint |
| Noir Profond | `#000000` | Primary backgrounds, contrast |
| Blanc Casse | `#F8F8F8` | Light backgrounds, purity, minimalism |

### Typography
- **Headings**: Playfair Display or Cormorant Garamond (Google Fonts, fallback for Luxia Display)
- **Body**: Lato, Montserrat Light, or Jost (sans-serif)
- **Body line-height**: 1.6–1.8, generous spacing throughout

## Architecture — Key Animation: "Product Branding Morph"

The centerpiece animation is a scroll-driven product transformation (unbranded white product → branded luxury product). Implemented via GSAP ScrollTrigger pinning and either:
- **Option A (preferred)**: Remotion pre-rendered image sequence (60–120 PNG frames), GSAP scrubs frame-by-frame on scroll
- **Option B**: CSS layers with opacity/clip-path animated via GSAP

## Site Sections (scroll order)

1. Loading Screen — SVG path-draw animation of dragonfly logo
2. Navigation — transparent, fixed
3. Hero — fullscreen, dark, product + golden mist + text reveal (GSAP SplitText)
4. Product Branding Morph — pinned scroll transformation (the showcase piece)
5. Storytelling — word-by-word scroll reveal
6. Values — 4 cards with staggered entrance
7. Collection — draggable horizontal carousel with 3D hover
8. Experience — horizontal scroll section (products in context)
9. Contact — minimal form with gold underline animations
10. Footer

## Animation Guidelines

- All animations must be **fluid and subtle** — never aggressive. Think perfume diffusing: slow, enveloping, memorable.
- GPU-accelerated only (`transform`, `opacity`). No layout-triggering animations.
- Respect `prefers-reduced-motion`: disable heavy animations, simplify Product Morph to a fade.
- Mobile: no custom cursor, drag instead of hover, reduce particles, simplify Product Morph.
- Custom cursor: thin gold circle (#C99700) following mouse with GSAP `quickTo` delay.

## Performance Requirements

- Images: WebP/AVIF with next/image lazy loading
- Code splitting per section
- Remotion exports: WebM + MP4 fallback, optimized file sizes
- Target: smooth 60fps on mid-range devices
